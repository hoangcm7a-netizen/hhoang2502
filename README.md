🚀 HHoang2502 - Personal Portfolio

Welcome to my personal portfolio repository! This project serves as my digital resume, showcasing my skills, projects, and creative media works. It is designed with a modern, developer-focused aesthetic (terminal/hacker theme) and built from the ground up using modern web technologies.

🌐 Live Demo: https://hhoang2502.netlify.app/

✨ Features

Interactive Terminal Intro: A custom typing animation that greets visitors in both Vietnamese and English.

Bilingual Content: Seamlessly displays project descriptions and about me information in both Vietnamese and English.

Project & Skill Showcase: Categorized view of my development projects (C#, React, .NET) and creative media works (CapCut, Premiere).

Anti-Spam Contact Form: A fully functional contact form powered by EmailJS, featuring a custom math-captcha challenge for repeated submissions to prevent spam.

Fully Responsive: Optimized for all devices, from desktop to mobile.

🛠️ Tech Stack

This project was built using the following modern ecosystem:

Framework: React + Vite

Language: TypeScript

Styling: Tailwind CSS

UI Components: shadcn/ui

Icons: Lucide React

Email Handling: EmailJS

💻 Running Locally

If you want to run this project on your local machine, follow these steps:

Prerequisites

Make sure you have Node.js installed on your machine.

Installation

Clone the repository:

git clone [https://github.com/YOUR_GITHUB_USERNAME/YOUR_REPO_NAME.git](https://github.com/YOUR_GITHUB_USERNAME/YOUR_REPO_NAME.git)


Navigate to the project directory:

cd YOUR_REPO_NAME


Install dependencies:

npm install


Start the development server:

npm run dev


Open http://localhost:5173 (or the port shown in your terminal) to view it in the browser.

🚀 Deployment

This project is optimized for deployment on platforms like Netlify or Vercel.
To deploy, simply link your GitHub repository to your hosting provider and set the build command to npm run build and the publish directory to dist.
